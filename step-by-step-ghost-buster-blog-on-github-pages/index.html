Step by step Ghost + Buster blog on Github pages

This is a step by step for Windows user but it's not much different for other Os's. If you never heard of Ghost well.. You're looking at it. It's a minimalistic CMS and I really like it so far.
Buster is a static page generator built for Ghost. We need it since you can only host static pages on Github.
So let's get started. We'll first download/install all the things we'll need:

  • Install Nodejs (version 0.10.x!)
  • Install Python 2.7
  • Get Ghost

We need pip to install Buster. Luckily for us Python 2.7 comes with easy install. So open up your terminal and run the following

cd "c:\Python27\Scripts"
easy_install.exe install pip

Now you should be able to

pip.exe install buster

Note that pip is located to the C:\Python27\Scripts folder.
At this point you should be able to run the'Node' and 'python' command, if Windows doesn't find it follow the next step.
It seems like Nodejs installer appends a trailing '/' in the Windows PATH variable and that cause it to fail. So open up control panel -> System -> Advanced Parameters -> Environments Variable then look for the Path variable and click edit.
You should have this included in your PATH:

C:\Program Files (x86)\nodejs;C:\Python27

Install Ghost

Let's get into the good stuff! Extract Ghost in the folder of your choice.
Get in there and run 

npm install --production
npm start

You can now access your blog locally on localhost:2368 and localhost:2368/ghost to quickly configurate it. Now that we have Ghost installed locally it's time to generate a static page and push it on Github.
So firstly go on Github and create a new repository nammed yourUsername.github.io
Get in the directory where you installed Buster and run the following command.

Generate static pages

buster.exe setup
buster.exe generate --domain 127.0.0.1:2368
buster.exe preview
buster.exe deploy

The setup command will ask you for your Github repository url.
The generate command simply generates static pages for you. Then the preview command will show you a preview of the generated pages and the deploy command will push everything on Github.
If you're using Cygwin forget about it it won't work. Also I got an issue with the buster executable but it also comes with a python script so use that instead if you have issues.
